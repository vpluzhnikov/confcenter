Анализ конфигураций:

Цель проекта - создать инструмент автоматической обработки конфигурационных выводов с оборудования. Сейчас в данном проекте реализована обработка и разбор выводов утилиты snap с серверов IBM Power. Результатом разбора является страница с данными о сервере (разделе сервера). Существует возможность выполнить экспорт конфигурации в PDF файл.

Примеры создаваемых отчетов:

  • Отчет из вывода snap с ключами gc (general information only)
  • Отчет из вывода snap с ключами ac (all information)
  • Промежуточный вывод в формате сайта для snap с ключами gc
  • Промежуточный вывод в формате сайта для snap с ключами ac
